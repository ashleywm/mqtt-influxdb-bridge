# MQTT Influxdb Bridge

Node.js implementation of a MQTT to Influxdb bridge for recording sensor data transmitted over MQTT to InfluxDB

## Installation

Yarn

```
yarn install
yarn run dev
```

npm 
```
npm install
npm run dev
```
